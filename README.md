# *Precognition*

Shoot'em up game made with opengl for the olcCodeJam2021

![](https://i.imgur.com/p4aq470.png)

# *Setup*

Open `start.bat` to generate the solution

# *Controls*

Movement - Arrow keys <br />
Shoot - X <br />
Focus - Shift
